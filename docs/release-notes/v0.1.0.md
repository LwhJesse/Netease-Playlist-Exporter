# v0.1.0

First public preview.

## Features

- Export NetEase Cloud Music playlists to CSV / JSON / import CSV / plain TXT
- Preserve original playlist order with an explicit `index` column
- Support QR login
- Support logged-in export
- Support anonymous export with warning
- Support `--expected-count` count verification
- Store login state and export output inside the project directory

## Known limitations

- Requires a compatible NetEase Cloud Music API service
- Anonymous exports may miss login-visible tracks
- API results may differ from the official web UI
- No packaged binary is provided
- This project exports metadata only; it does not download audio
